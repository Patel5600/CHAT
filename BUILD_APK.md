# SEAL Chat - Complete APK Build Guide\n\nThis guide provides step-by-step instructions to build the SEAL Chat APK on your local machine.\n\n## Prerequisites\n\n### System Requirements\n- **Windows:** Windows 10 or later (8GB RAM minimum)\n- **Mac:** macOS 10.13 or later (8GB RAM minimum)\n- **Linux:** Ubuntu 18.04 or later (8GB RAM minimum)\n- **Disk Space:** 10GB free space\n\n### Required Software\n1. **Java Development Kit (JDK) 11+**\n2. **Android Studio**\n3. **Git**\n\n---\n\n## Step 1: Install Java Development Kit (JDK)\n\n### Windows\n1. Download JDK 11+ from https://www.oracle.com/java/technologies/downloads/\n2. Run the installer\n3. Follow the installation wizard\n4. Verify installation:\n   ```bash\n   java -version\n   ```\n\n### Mac\n```bash\n# Using Homebrew (recommended)\nbrew install openjdk@11\n\n# Verify\njava -version\n```\n\n### Linux (Ubuntu)\n```bash\nsudo apt-get update\nsudo apt-get install openjdk-11-jdk\n\n# Verify\njava -version\n```\n\n---\n\n## Step 2: Install Android Studio\n\n### All Platforms\n1. Download Android Studio from https://developer.android.com/studio\n2. Run the installer\n3. Follow the setup wizard\n4. When prompted, install:\n   - Android SDK\n   - Android SDK Build-Tools\n   - Android Emulator\n   - Android SDK Platform-Tools\n\n### First Launch Setup\n1. Open Android Studio\n2. Go to **File** → **Settings** (Windows/Linux) or **Android Studio** → **Preferences** (Mac)\n3. Navigate to **Appearance & Behavior** → **System Settings** → **Android SDK**\n4. Click **SDK Platforms** tab\n5. Select **Android 14** (or latest)\n6. Click **Apply** → **OK**\n\n---\n\n## Step 3: Clone the Repository\n\n### Windows (Command Prompt or PowerShell)\n```bash\ncd Documents\ngit clone https://github.com/Patel5600/CHAT.git\ncd CHAT\n```\n\n### Mac/Linux (Terminal)\n```bash\ncd ~\ngit clone https://github.com/Patel5600/CHAT.git\ncd CHAT\n```\n\n---\n\n## Step 4: Build the APK\n\n### Option A: Using Android Studio (Easiest)\n\n1. **Open Project**\n   - Launch Android Studio\n   - Click **File** → **Open**\n   - Navigate to the `CHAT` folder\n   - Click **OK**\n   - Wait for Gradle sync to complete\n\n2. **Build APK**\n   - Click **Build** in the menu bar\n   - Select **Build Bundle(s) / APK(s)**\n   - Click **Build APK(s)**\n   - Wait for the build to complete (2-5 minutes)\n\n3. **Locate APK**\n   - A notification will appear: \"Build successful\"\n   - Click **Locate** or navigate to:\n     ```\n     CHAT/app/build/outputs/apk/debug/app-debug.apk\n     ```\n\n### Option B: Using Command Line (Faster)\n\n#### Windows (Command Prompt)\n```bash\ncd CHAT\ngradlew.bat assembleDebug\n```\n\n#### Mac/Linux (Terminal)\n```bash\ncd CHAT\nchmod +x gradlew\n./gradlew assembleDebug\n```\n\n**Output:** `app/build/outputs/apk/debug/app-debug.apk`\n\n### Option C: Build Release APK (For Production)\n\n#### Windows\n```bash\ncd CHAT\ngradlew.bat bundleRelease\n```\n\n#### Mac/Linux\n```bash\ncd CHAT\n./gradlew bundleRelease\n```\n\n**Output:** `app/build/outputs/bundle/release/app-release.aab`\n\n---\n\n## Step 5: Install APK on Device/Emulator\n\n### Using Android Studio\n\n1. **Connect Device**\n   - Enable USB Debugging on your Android phone\n   - Connect via USB cable\n\n2. **Run App**\n   - Click **Run** in the menu bar\n   - Select your device\n   - Click **OK**\n   - App will install and launch automatically\n\n### Using Command Line\n\n#### Windows\n```bash\ncd CHAT\nadb install app\\build\\outputs\\apk\\debug\\app-debug.apk\n```\n\n#### Mac/Linux\n```bash\ncd CHAT\nadb install app/build/outputs/apk/debug/app-debug.apk\n```\n\n---\n\n## Troubleshooting\n\n### \"Gradle sync failed\"\n- **Solution:** Click **File** → **Sync Now**\n- If still failing, delete `.gradle` folder and retry\n\n### \"Build failed: SDK not found\"\n- **Solution:** Go to **File** → **Settings** → **Android SDK**\n- Ensure Android SDK is installed\n- Click **Apply** → **OK**\n\n### \"Java version mismatch\"\n- **Solution:** Ensure JDK 11+ is installed\n- Verify: `java -version`\n- Update `JAVA_HOME` environment variable if needed\n\n### \"adb: command not found\"\n- **Solution:** Add Android SDK tools to PATH\n- Windows: `C:\\Users\\YourName\\AppData\\Local\\Android\\Sdk\\platform-tools`\n- Mac/Linux: `~/Android/Sdk/platform-tools`\n\n### Build takes too long\n- **Solution:** This is normal for first build (5-10 minutes)\n- Subsequent builds are faster (1-2 minutes)\n- Ensure you have 10GB free disk space\n\n---\n\n## Testing the App\n\n### Demo Credentials\n- **Email:** test@example.com\n- **OTP:** 123456\n\n### Features to Test\n1. **Login Screen**\n   -
